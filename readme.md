# Python for Data Science and Statistical Computing

This intensive two-day workshop provides a comprehensive introduction to Python for researchers and data analysts who are new to Python but need to perform data analysis, statistical computing, and scientific computing tasks.
The course covers Python fundamentals, programming essentials, and the core Python data science stack, providing participants with the skills to transition from other data analysis tools (such as R, MATLAB, or Excel) to Python.
By combining general Python programming with hands-on practice using NumPy, Pandas, and data visualization libraries, participants will gain a solid foundation for pursuing machine learning and deep learning in Python.

#### Day 1: Python Fundamentals and Programming

**Session 1 (2 hours) - Getting Started with Python**

- Installing Python: virtual environments, pip, conda
- Integrated Development Environments: Jupyter notebooks, Spyder, PyCharm, Positron
- Elementary data types: integers, floats, booleans, strings
- String operations, methods, and indexing
- Basic input/output and working with the interactive interpreter

**Session 2 (2 hours) - Collections and Data Structures**

- Lists: creation, indexing, slicing, methods, nested lists
- Dictionaries: key-value pairs, methods, iteration
- Tuples: immutability, unpacking, use cases
- Sets: unique elements, set operations, when to use sets
- Type conversion and choosing the right data structure

**Session 3 (2 hours) - Programming Essentials**

- Control flow: if/elif/else conditionals
- Loops: for loops, while loops, loop control (break, continue)
- List comprehensions, dictionary comprehensions, set comprehensions
- Functions: defining functions, parameters, default arguments, return values
- Lambda functions and functional programming basics
- Working with enumerate and zip

#### Day 2: The Data Science Stack

**Session 1 (2 hours) - Numerical Computing with NumPy**

- Introduction to NumPy arrays: creation, data types, performance advantages
- Array operations: element-wise operations, broadcasting, aggregations
- Indexing and slicing: boolean indexing, fancy indexing, reshaping
- Linear algebra operations: dot products, matrix multiplication
- Performance comparison: NumPy vs pure Python
- Working with multidimensional arrays

**Session 2 (2 hours) - Data Processing with Pandas**

- Introduction to DataFrames and Series
- Creating DataFrames from various sources (CSV, dictionaries, arrays)
- Selecting and filtering: row/column selection, boolean filtering, loc/iloc
- Data manipulation: adding columns, sorting, handling missing data
- Groupby operations: split-apply-combine paradigm
- Basic aggregations and summary statistics
- Introduction to data wrangling workflows

**Session 3 (2 hours) - Visualization and Statistical Analysis**

- Data visualization fundamentals with Matplotlib
- Statistical visualizations with Seaborn: distributions, relationships, categorical data
- Grammar of graphics approach with plotnine (ggplot for Python)
- Introduction to statistical modeling with statsmodels
- Basic linear regression and model interpretation
- Working with modules and packages: importing, creating custom modules
- Overview of the broader Python ecosystem for data science

### Format

- 2 days, 6 sessions (2 hours each)
- Hands-on workshop with extensive live coding
- Python scripts or Jupyter notebooks for all examples and exercises
- Participants write and execute code throughout the course
- Real datasets from scientific research and data analysis
- Mix of instructor demonstrations and guided practice
- All code and materials provided for independent learning after the course

### Learning Outcomes

By the end of this course, participants will be able to:

- Install and configure Python environments for data science work
- Write Python programs using core language features (variables, functions, loops, conditionals)
- Choose and use appropriate data structures for different tasks
- Perform efficient numerical computations with NumPy arrays
- Load, manipulate, and analyze datasets using Pandas DataFrames
- Create publication-quality visualizations of data
- Understand the structure and philosophy of the Python data science ecosystem
- Read and modify Python code written by others
- Know where to find resources and packages for continued learning
- Have a solid foundation for pursuing machine learning and deep learning with Python

### What This Course Does *Not* Cover

- Machine learning algorithms (covered in separate courses)
- Deep learning and neural networks (covered in separate course)
- Advanced statistical methods (Bayesian analysis, mixed effects models, etc.)
- Production deployment and software engineering practices
- Web scraping, APIs, or database access (mentioned briefly)
- Advanced NumPy/Pandas features (covered through practice)
- Object-oriented programming in depth (introduced but not comprehensive)
