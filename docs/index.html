Python for Data Science

A Two-Day Workshop

Author

Mark Andrews

Published

7 April, 2026

This workshop covers Python for scientific computing and data analysis. The first day introduces the Python language itself: its data model, built-in types, collection structures, and core programming constructs. The second day covers the data science stack: NumPy for numerical computing, Pandas for data manipulation, and tools for visualization and statistical modelling.

Day One

Session One: Numbers, Booleans, and Strings

We begin with an overview of Python’s object model, covering the concepts of objects, name binding, namespaces, and garbage collection. We then work through Python’s scalar types: integers, floating-point numbers, Booleans, and strings. For each type we examine how objects are created and bound to names, what operators apply to them, and which built-in functions are available. For strings, we also cover f-string formatting, indexing and slicing, and string methods. The session closes with an overview of type conversion between the scalar types.

Session Two: Collections

This session covers Python’s built-in collection types. We work through lists, dictionaries, tuples, and sets, examining how each is created, how its elements are accessed and modified, and what operations and methods apply to it. The session closes with an overview of type conversion between collection types.

Session Three: Iteration, Functions, and Classes

This session introduces the core programming constructs needed for practical Python work. We begin with iteration using for and while loops, then cover list, dictionary, and set comprehensions as a more concise alternative. We proceed to defining functions, covering parameters, default arguments, variable-length argument lists, and lambda expressions together with map and filter. Conditional statements come next, shown primarily inside functions where they are most natural. The session closes with an introduction to classes and object-oriented programming.

Day Two

Session One: Numerical computing with NumPy

This session begins with Python’s import system, covering how to use modules from the standard library and third-party packages. We then introduce NumPy, the foundational library for numerical computing in Python. We cover array creation and data types, element-wise operations and broadcasting, mathematical functions, and aggregations including axis-wise reductions on multidimensional arrays. The session closes with indexing and boolean indexing, array reshaping, and an introduction to matrix operations.

Session Two: Pandas

This session introduces Pandas, the primary library for data manipulation in Python. We begin with the two fundamental data structures, Series and DataFrame, and cover how to load data from CSV files. We then cover the core operations on DataFrames: selecting columns with square brackets, loc, and iloc; renaming columns; filtering rows with boolean conditions and query; adding and transforming columns with assign; sorting; and the split-apply-combine pattern using groupby and aggregation.

Session Three: Visualization and Statistical Modelling

This session covers data visualization and statistical modelling in Python. We begin with Matplotlib, the foundational plotting library, covering the pyplot interface, common plot types, and the object-oriented API for multi-panel figures. We then introduce Seaborn, which builds on Matplotlib to produce statistical graphics from tidy data with minimal code. The third topic is plotnine, a Python implementation of the grammar of graphics that closely mirrors R’s ggplot2. The session closes with statistical modelling using statsmodels, covering linear regression, logistic regression, and linear mixed-effects models via the formula API, together with a brief survey of the broader Python statistical modelling ecosystem.
